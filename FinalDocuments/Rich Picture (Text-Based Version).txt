Used case diagram
Actors
* Student

* Teacher

* Admin

* AI Engine

Subsystems
   1. Personalized Learning Module

   2. Assessment & Feedback Module

   3. Teacher Analytics Dashboard

   4. Career Support Module

   5. Database & Security Layer

Use Cases
Student:
 - View recommended lessons
 - Take assessments
 - Receive AI feedback
 - View skill progress
 - Get career suggestions


Teacher:
 - Monitor class analytics
 - Adjust content
 - View at-risk students
 - Review assessment insights


Admin:
 - Manage users
 - Configure system rules
 - Oversee data governance




Complete Sequence Diagram


Student → System UI: Login
System UI → Database: Verify credentials
Database → System UI: Auth success


Student → Learning Module: Request recommendations
Learning Module → AI Engine: Analyze student profile
AI Engine → Database: Fetch history, performance data
Database → AI Engine: Return student data
AI Engine → Learning Module: Generate recommendations
Learning Module → Student: Display personalized learning path
