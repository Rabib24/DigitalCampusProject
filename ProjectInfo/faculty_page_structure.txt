Faculty Dashboard Page Structure Plan
===================================

Based on FacultyDashboard.txt specification, here's the detailed page structure:

1. MAIN DASHBOARD PAGES
=====================
1.1. Faculty Dashboard (Main) - /faculty/
- Header with user info and quick actions
- Overview widgets:
  - Course summary (current courses, enrollment counts)
  - Recent assignments (due soon, needing grading)
  - Upcoming classes/schedule
  - Recent announcements
  - Quick action buttons (create assignment, post announcement, etc.)
  - Research project status
  - Advisee alerts (if any)

1.2. Course List Page - /faculty/courses/
- Filterable/searchable list of courses
- Course cards with key info (title, code, semester, enrollment)
- Quick actions per course (view details, gradebook, assignments)
- Create new course button

1.3. Schedule Page - /faculty/schedule/
- Weekly calendar view
- Class sessions with details
- Recording status indicators
- Quick navigation to class materials

1.4. Notifications Page - /faculty/notifications/
- List of all notifications
- Filtering by type (announcements, assignments, etc.)
- Mark as read/unread functionality
- Notification settings

1.5. Announcements Page - /faculty/announcements/
- List of posted announcements
- Create new announcement button
- Filter by course, date, status
- Edit/delete functionality

2. COURSE MANAGEMENT PAGES
========================
2.1. Course Creation Page - /faculty/courses/create/
- Form for course details (title, code, description, etc.)
- Semester selection
- Prerequisites configuration
- Enrollment settings
- Syllabus editor

2.2. Course Details Page - /faculty/courses/[id]/
- Course overview
- Navigation tabs to related sections
- Quick actions (edit, delete, clone)
- Enrollment statistics

2.3. Course Syllabus Page - /faculty/courses/[id]/syllabus/
- Syllabus content display
- Edit button for faculty
- Version history
- Download/print options

2.4. Course Notes Page - /faculty/courses/[id]/notes/
- List of lecture notes
- Upload new notes
- Organize by date/topic
- Search functionality

2.5. Course Assignments Page - /faculty/courses/[id]/assignments/
- List of assignments for this course
- Create new assignment button
- Assignment statistics
- Quick grading access

2.6. Course Gradebook Page - /faculty/courses/[id]/gradebook/
- Student roster with grades
- Assignment columns
- Export functionality
- Grade statistics
- Manual grade entry

2.7. Course Discussion Forum Page - /faculty/courses/[id]/forum/
- List of forum threads
- Create new thread
- Moderate posts
- Search functionality

2.8. Class Recording Management Page - /faculty/courses/[id]/recordings/
- List of recordings
- Recording controls (start/stop)
- Privacy settings
- Download/share options

2.9. Attendance Tracking Page - /faculty/courses/[id]/attendance/
- Session list
- Attendance marking interface
- Reports and analytics
- Export functionality

3. ASSIGNMENT & GRADING PAGES
===========================
3.1. Assignment Creation Page - /faculty/assignments/create/
- Assignment details form
- Due date and time
- Submission type (file, text, etc.)
- Rubric builder
- File attachments

3.2. Assignment Details Page - /faculty/assignments/[id]/
- Assignment overview
- Submission statistics
- Quick actions (edit, delete, grade)
- Student submission list

3.3. Grading Page (for faculty) - /faculty/assignments/[id]/grade/
- Student submission viewer
- Grade entry form
- Feedback editor
- Rubric application
- Save/submit grades

3.4. Plagiarism Report Page - /faculty/assignments/[id]/plagiarism/
- Plagiarism detection results
- Detailed similarity reports
- Flagged content highlighting
- Action buttons (review, approve, etc.)

3.5. Submission Review Page - /faculty/assignments/[id]/submissions/[submissionId]/
- Student submission content
- Grade and feedback display
- Revision history
- Download options

4. ADVISING PAGES (Faculty as Advisors)
=====================================
4.1. Advisee List Page - /faculty/advising/advisees/
- List of assigned advisees
- Key metrics per student (GPA, credits, etc.)
- Alert indicators (low GPA, missing requirements)
- Quick access to student details

4.2. Advisee Progress Tracking Page - /faculty/advising/advisees/[id]/progress/
- Academic history
- Degree progress tracking
- Course recommendations
- Retake suggestions
- Milestone tracking

4.3. Appointment Scheduling Page - /faculty/advising/appointments/
- Calendar view
- Available time slots
- Appointment requests
- Meeting history

4.4. CGPA Simulator Page - /faculty/advising/cgpa-simulator/
- Interactive GPA calculator
- "What-if" scenario planning
- Course selection interface
- Results visualization

4.5. Milestone Reminder Page - /faculty/advising/milestones/
- List of upcoming milestones
- Student progress tracking
- Alert system
- Reminder configuration

4.6. Retake Recommendation Page - /faculty/advising/retake-recommendations/
- Course retake suggestions
- Academic improvement plans
- Prerequisite analysis
- Schedule optimization

5.3. Project Proposal Submission Page - /faculty/research/proposals/create/

5.4. Grant Application Page - /faculty/research/grants/apply/

5.5. Publication Management Page - /faculty/research/publications/

5.6. Ethics Approval Page - /faculty/research/ethics/
- Ethics application forms
- Submission tracking
- Review status
- Document repository
